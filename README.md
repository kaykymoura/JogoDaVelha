# ❌⭕ Jogo da Velha

Um simples **Jogo da Velha** desenvolvido em **HTML, CSS e JavaScript**.  
O projeto foi feito para praticar lógica de programação e manipulação do DOM.  

---

## 🚀 Tecnologias
- HTML5  
- CSS3  
- JavaScript  

---

## 📌 Funcionalidades
- Dois jogadores (local)  
- Verificação automática de vencedor  
- Reiniciar partida  

---
